# :spider: Araknomecha Scrapper

This project contains all the code used to gather Wakfu data to be provided to [Corvo Astral](https://github.com/Markkop/corvo-astral).  
Main sources are: 
* Wakfu's [CDN](https://www.wakfu.com/en/forum/332-development/236779-json-data)
* Wakfu's [Website](https://www.wakfu.com/en/mmorpg/)
* [Method](https://builder.methodwakfu.com/builder/main)'s public API. 

It's still not organizated and integrated directly with Corvo Astral.
